## "He Who Consumes" Animated Cosmic Horror Scene Assets

These are all the assets I created when making a horror short called "He Who Consumes". I used Blender to model a Galactus Helmet, a boat, a beach chair, and a tent. I then used Unreal Engine to animate the movements and create the entire scene.
